На основі опису фічі згенеруй 5-10 тест-кейсів.

Опис фічі:
{{feature}}

Вимоги до тест-кейсів:
1. Включи позитивні, негативні та граничні сценарії
2. Кожен тест-кейс повинен мати: id, title, preconditions, steps, expected, priority
3. Познач пріоритет одним із: High, Medium, Low

Поверни валідний JSON такого формату:
{
  "cases": [
    {
      "id": "TC-01",
      "title": "...",
      "preconditions": ["..."],
      "steps": ["..."],
      "expected": ["..."],
      "priority": "High"
    }
  ]
}
